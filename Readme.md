# rust-backup is a tool created to keep backups.
## It parses the paths of the items that are to be saved from data.json
## It saves them to the "Export_Path" you give it in the json.

## "file": "a" = the filename or a short description, optional.
## "current_file_path": "C:/Users/User/Documents/test.exe" = the path for the file you want to keep a backup.
## "export_path": "C:/Users/User/Documents/Backups" where to keep the backup.
## "create_new_folder": 1 = decide if it will create a new folder with date, hour and minute timestamp or not.
